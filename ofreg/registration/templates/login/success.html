Welcome to the Ofelia Federation

In order to use the configuration web application and to enter the experiment network of Ofelia, please download the OpenVPN configuration and fire up your OpenVPN.

The minimum requirements for the tunnel to work is OpenVPN 2.x.x and optionally resolvconf to update your DNS (e.g. sudo apt-get install resolvconf).
Please refer to the documentation wiki for setup.

After opening the tunnel, please visit Expedient. Please see the wiki for links and more information.

Please also see the official Ofelia website.
