Help

OFELIA provides various sources of help:

Documentation

  • OFELIA user documentation: information about the facility and how to work with the OFELIA Control Framework from an experimenter point of view.
  • Video tutorials

How to set up slice and project

How to configure experiment

Helpdesk

  • OFELIA helpdesk: meant to provide experimenters a way to report and open trouble tickets where the OFELIA user documentation is not enough.

Mailing lists

  • Users discuss mailing List: discussions between users and Island Managers regarding any problem or doubt that could not be solved with the documentation. Every OFELIA user is automatically registered to this list.
  • Developers mailing List: technical discussion about development within the OFELIA project. If you have questions or would like to contribute to OFELIA community, this is the place.
